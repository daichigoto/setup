# 3押し高速オープン設定【Firefox】

- ☆→「保存」→「Ctrl」＋「Shift」＋「O」→対象ブックマークを選択

|項目|値|
|:---|:---|
|名前|名前（任意）|
|URL|高速オープンしたいページのURL|
|タグ|任意|
|キーワード|アルファベット。短縮入力に使う文字。可能であれば1文字|

# 使い方

- Windows 「Ctrl」＋「L」「キーワード」「Enter」
- Mac 「⌘」＋「L」「キーワード」「Enter」
