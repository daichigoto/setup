# FreeBSD新規インストール時 セットアップ (執筆環境)

## PATH

    ~/bin を追加
    ~/Documents/tttcmds/bin を追加
    ~/Documents/misc/bin を追加

## tttcmds

    mkdir -p ~/Documents
    cd ~/Documents/
    git clone git@github.com:daichigoto/tttcmds.git
    cd tttcmds
    make

## misc

    mkdir -p ~/Documents
    cd ~/Documents/
    git clone git@github.com:daichigoto/misc.git
