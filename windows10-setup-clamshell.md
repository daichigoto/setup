# クラムシェルモード セットアップ

1. 設定アプリケーション：「システム」→「電源とスリープ」→次の時間が経過後、PCをスリープ状態にする(電源に接続時)→「なし」
2. 設定アプリケーション：「システム」→「電源とスリープ」→「電源の追加設定」
3. 電源オプション：「カバーを閉じたときの動作の選択」→電源に接続×カバーを閉じたときの動作「何もしない」→「変更の保存」
4. ノートPCのカバーを閉じる
